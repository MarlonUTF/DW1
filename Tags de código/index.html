Tags de código

Apresentar blocos de código em documentos HTML tornou-se uma atividade bastante comum, visto que existem inúmeros sites que precisam exigir trechos de código. E não apenas código, também precisamos mostrar a saída do código ao tentar replicar janelas de terminais

Blogs de tecnologia, plataformas de ensino de programação, documentações de software, provedores de API – há muitos contextos em que ser capaz de apresentar código ao usuário é um requisito vital.

Tag code

Ela por si só representa apenas uma única frase de código ou linha de código.

A code tag é usada para definir um trecho de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador:

Ao estiliza-la, fornece uma melhor visualização a usuário; Exemplo:

.code {
background-color: #eee;
border-radius: 3px;
font-family: courier, monospace;
padding: 0 3px;
color: red;
}

Tag samp

A tag samp é usada para definir a saída de amostra de um programa de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador. Exemplo:

File not found.
Press F1 to continue

Tag pre

HTML texto preformatado pre; é a tag utilizada para representar texto pré-formatado. Um texto dentro desse elemento é tipicamente exibido em uma fonte não proporcional da mesma maneira em que o texto original foi disposto no arquivo. Espaços em branco são mantidos no texto da mesma forma em que este foi digitado. Exemplo: 

            body {
              color:red;
            }

Tag kbd

A tag kbd é usada para definir a entrada do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador. Exemplo:

<p>Press Ctrl + C to copy text (Windows).</p>

Press Cmd + C to copy text (Mac OS).
